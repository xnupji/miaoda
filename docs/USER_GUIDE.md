# HTP挖矿平台使用指南

## 平台简介

HTP挖矿平台是一个基于BSC区块链的去中心化挖矿网站，用户通过完成简单任务挖矿获得HTP代币奖励，支持邀请机制和主节点申请推广系统。

## 主要功能

### 1. 用户系统
- **注册登录**：使用用户名和密码注册登录
- **邀请机制**：使用邀请码注册可获得奖励
- **角色系统**：普通用户、主节点、管理员三种角色

### 2. 挖矿功能
- **每日签到挖矿**：每天登录点击即可获得5-15 HTP代币奖励
- **挖矿记录**：查看历史挖矿记录和收益统计
- **自动发放**：挖矿奖励自动发放到账户

### 3. 邀请系统
- **邀请码生成**：每个用户拥有唯一邀请码
- **邀请奖励**：邀请好友注册获得10 HTP奖励
- **邀请统计**：查看邀请人数和累计奖励

### 4. 主节点系统
- **申请主节点**：完成10万个钱包地址激活任务
- **激活奖励**：每激活1个地址奖励3 USDT
- **进度追踪**：实时查看激活进度和奖励统计

### 5. 钱包管理
- **Metamask连接**：连接Metamask钱包到BSC网络
- **地址绑定**：绑定钱包地址用于接收代币
- **提币申请**：申请将代币转出到钱包地址
- **审核机制**：提币需要管理员审核

### 6. 管理后台（仅管理员）
- **用户管理**：查看所有用户，管理用户角色
- **提币审核**：审核用户的提币申请
- **主节点审核**：审核主节点申请
- **数据统计**：查看平台整体数据

## 使用流程

### 新用户注册
1. 访问登录页面
2. 切换到"注册"标签
3. 填写用户名、密码（可选填邀请码）
4. 点击注册按钮
5. 自动登录进入平台

### 开始挖矿
1. 安装Metamask钱包扩展
2. 在"钱包管理"页面连接Metamask
3. 绑定钱包地址
4. 返回"仪表盘"页面
5. 点击"开始挖矿"按钮
6. 每天可挖矿一次

### 邀请好友
1. 进入"邀请好友"页面
2. 复制您的邀请码或邀请链接
3. 分享给好友
4. 好友使用邀请码注册后，您将获得10 HTP奖励

### 申请主节点
1. 确保已绑定钱包地址
2. 进入"主节点"页面
3. 点击"提交申请"
4. 等待管理员审核
5. 审核通过后开始推广任务

### 提币操作
1. 进入"钱包管理"页面
2. 点击"申请提币"
3. 选择代币类型（HTP或USDT）
4. 填写提币数量和接收地址
5. 如果是HTP提币，需填写已支付的USDT金额
6. 提交申请等待管理员审核

## 重要说明

### 代币说明
- **HTP代币**：平台虚拟代币，通过挖矿和邀请获得
- **USDT**：虚拟USDT，用于支付HTP提币费用和主节点奖励
- **注意**：所有代币均为虚拟发行，非真实区块链代币

### 提币规则
- HTP代币转出需要向开发者转入相应数量的虚拟USDT
- 所有提币申请需要经过管理员审核
- 审核通过后代币将转入您的钱包地址

### 主节点规则
- 需要完成100,000个钱包地址激活
- 每激活1个地址奖励3 USDT
- 申请需要管理员审核通过

### 安全提示
- 妥善保管您的登录密码
- 确保钱包地址正确无误
- 不要泄露您的私钥信息

## 技术架构

- **前端框架**：React + TypeScript + Vite
- **UI组件**：shadcn/ui + Tailwind CSS
- **后端服务**：Supabase
- **区块链**：BSC (Binance Smart Chain)
- **钱包集成**：Metamask + ethers.js

## 管理员功能

第一个注册的用户自动成为管理员，管理员可以：
- 查看所有用户信息
- 管理用户角色
- 审核提币申请
- 审核主节点申请
- 查看平台统计数据
