# HTP挖矿平台需求文档

## 1. 网站概述

### 1.1 网站名称

HTP挖矿平台

### 1.2 网站描述

基于BSC区块链的去中心化挖矿网站，用户通过完成简单任务挖矿获得HTP代币奖励，支持邀请机制和主节点申请推广系统。\n
## 2. 核心功能

### 2.1 用户挖矿功能

- 用户每日签到挖矿：用户每天登录网站点击即可获得HTP代币奖励
- 挖矿记录查看：用户可查看历史挖矿记录和收益

### 2.2 代币系统

- 代币名称：HTP
- 代币类型：自定义ERC20代币（虚拟发行）\n- 发行方式：通过用户挖矿产生
- 代币余额查询：实时显示用户HTP余额

### 2.3 价格显示机制

- 价格展示：实时显示HTP代币价格
- 价格计算规则：价格按照每天递增0.03的规则计算，开盘价设定为0.01
- 价格更新频率：每日自动更新一次

### 2.4 审核机制

- 交互地址审核：用户用在网站所创建的钱包，每天都可以点击挖矿产生代币。发出一定数量，用户想把代币转出。需要向开发者转入相应的虚拟的USDT。用户钱包地址所产生的代币需要经过管理员审核才能进行转出操作
- 审核状态显示：用户可查看审核进度和结果

### 2.5 邀请机制

- 邀请码生成：用户可生成个人邀请码
- 邀请注册：新用户可通过邀请码注册
- 邀请奖励：邀请他人注册可获得HTP奖励
- 下线统计：显示邀请的下线用户数量

### 2.6 主节点系统

- 主节点申请：用户可申请成为主节点,帮开发者完成相应的用户积载,主节点要求完成10万个钱包地址，并且激活开发者会给用户申请主节点的人一定的奖励。一个地址激活奖励3个虚拟usDT。\n- 邀请码管理：主节点可生成专属邀请码
- 推广任务：完成指定推广任务获得HTP奖励
- 主节点面板：专属管理界面

### 2.7 防作弊机制

- 人工验证：通过简单的人机验证确保是真实用户操作
- 挖矿频率无限制：可以自动化脚本刷取
- 钱包地址绑定：用户需要绑定个人钱包地址接收代币

### 2.8 区块链集成

- 区块链网络：BSC（币安智能链）\n- 合约部署：部署HTP代币合约和挖矿逻辑合约
- 钱包连接：支持Metamask等主流钱包连接
- 交易记录：显示用户的HTP转账记录

### 2.9 投资方展示

- 投资方列表：展示项目投资方信息
- 投资方名称：a16z、Eden Block、Galaxy Digital、AMAVEN11、Maven11、CoinFund、Protocol Labs、Jsquare、M31 Capita、Peer VC、币安、OKX、Gate.io
- 融资金额：3000万美金
- 开发时间：2023年开始投入开发
- 投资方详情页面：提供投资方详细介绍链接

## 3. 管理后台功能

### 3.1 管理员登录

- 管理员专用登录入口
- 多重身份验证机制
- 仅限开发者和授权管理员访问

### 3.2 用户管理

- 注册用户列表：查看所有用户基本信息
- 挖矿信息管理：监控和管理用户挖矿记录
- 邀请关系查看：查看用户的邀请人和下线数量
- 权益管理：调整用户权限和收益分配

### 3.3 开发者钱包管理

- 签约地址设置：在后台更改开发者钱包地址
- 地址验证机制：确保钱包地址安全有效
- 权限控制：只有开发者和后台登录者有权更改钱包地址
- 地址变更记录：记录历次钱包地址更改历史

### 3.4 数据统计

- 用户统计数据：总用户数、活跃用户数
- 挖矿数据统计：总产量、日产量
- 邀请数据统计：总邀请数、邀请分布
- 收益分配统计：各层级用户收益情况

## 4. 网站设计风格

### 4.1 配色方案

主色调采用科技蓝（#1F7AFF）搭配深空灰（#1C1C24），辅以渐变光效营造科幻感

### 4.2 视觉细节

- 采用圆角卡片式设计，圆角半径8px
- 添加发光边框效果和渐变背景
- 图标采用线性风格，配合微动画效果
- 按钮具有悬停发光和点击脉冲动画

### 4.3 整体布局

采用卡片式布局，功能模块清晰分区，左侧导航栏固定，右侧内容区域响应式设计
