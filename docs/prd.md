# Web3.0挖矿平台需求文档

## 1. 网站概述

### 1.1 网站名称

HTP挖矿平台

### 1.2 网站描述

基于BSC区块链的去中心化挖矿网站，用户通过完成简单任务挖矿获得HTP代币奖励，支持邀请机制和主节点申请推广系统。\n

## 2. 核心功能

### 2.1 用户挖矿功能

- 用户每日签到挖矿：用户每天登录网站点击即可获得HTP代币奖励
- 挖矿记录查看：用户可查看历史挖矿记录和收益

### 2.2 代币系统

- 代币名称：HTP
- 代币类型：自定义ERC20代币（虚拟发行）\n- 发行方式：通过用户挖矿产生
- 代币余额查询：实时显示用户HTP余额

### 2.3 审核机制

- 交互地址审核：用户用在网站所创建的钱包，每天都可以点击挖矿产生代币。发出一定数量，用户想把代币转出。需要向开发者转入相应的虚拟的USDT。用户钱包地址所产生的代币需要经过管理员审核才能进行转出操作
- 审核对话框：开发专门的审核界面和对话框功能
- 审核状态显示：用户可查看审核进度和结果

### 2.4 邀请机制

- 邀请码生成：用户可生成个人邀请码
- 邀请注册：新用户可通过邀请码注册
- 邀请奖励：邀请他人注册可获得HTP奖励
- 下线统计：显示邀请的下线用户数量

### 2.5 主节点系统

- 主节点申请：用户可申请成为主节点,帮开发者完成相应的用户积载,主节点要求完成10万个钱包地址，并且激活开发者会给用户申请主节点的人一定的奖励。一个地址激活奖励3个虚拟usDT。
- 邀请码管理：主节点可生成专属邀请码
- 推广任务：完成指定推广任务获得HTP奖励
- 主节点面板：专属管理界面

### 2.6 防作弊机制

- 人工验证：通过简单的人机验证确保是真实用户操作
- 挖矿频率无限制：可以自动化脚本刷取
- 钱包地址绑定：用户需要绑定个人钱包地址接收代币

### 2.7 区块链集成

- 区块链网络：BSC（币安智能链）\n- 合约部署：部署HTP代币合约和挖矿逻辑合约
- 钱包连接：支持Metamask等主流钱包连接
- 交易记录：显示用户的HTP转账记录

## 3. 网站设计风格

### 3.1 配色方案

主色调采用科技蓝（#1F7AFF）搭配深空灰（#1C1C24），辅以渐变光效营造科幻感

### 3.2 视觉细节

- 采用圆角卡片式设计，圆角半径8px
- 添加发光边框效果和渐变背景
- 图标采用线性风格，配合微动画效果
- 按钮具有悬停发光和点击脉冲动画

### 3.3 整体布局

采用卡片式布局，功能模块清晰分区，左侧导航栏固定，右侧内容区域响应式设计
