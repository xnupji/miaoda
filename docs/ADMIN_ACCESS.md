## 🔑 如何看到管理后台入口

### 方法一：刷新页面（推荐）
1. 在当前页面按 `F5` 或 `Ctrl+R` (Windows) / `Cmd+R` (Mac) 刷新页面
2. 页面刷新后，左侧导航栏底部会出现"管理后台"入口（盾牌图标）

### 方法二：重新登录
1. 点击右上角用户头像
2. 选择"退出登录"
3. 重新使用 `chfyan@miaoda.com` 登录
4. 登录后左侧导航栏底部会显示"管理后台"入口

### 方法三：直接访问URL
1. 在浏览器地址栏输入：`/admin`
2. 直接访问管理后台页面

## 📍 管理后台入口位置

管理后台入口位于：
- **位置**：左侧导航栏底部
- **显示**：在"钱包管理"下方，有一条分隔线
- **图标**：盾牌图标（🛡️）
- **文字**：管理后台

## 🔐 权限说明

### 管理员（您的账户）
- ✅ 可以看到"管理后台"入口
- ✅ 可以访问 `/admin` 页面
- ✅ 可以管理所有用户
- ✅ 可以审核提币和主节点申请

### 主节点用户

- ❌ 无法访问 `/admin` 页面
- ✅ 可以使用其他所有功能

### 普通用户
- ❌ 看不到"管理后台"入口
- ❌ 无法访问 `/admin` 页面
- ✅ 可以使用基础功能

## 🎯 管理后台功能

进入管理后台后，您可以：

### 1. 用户管理
- 查看所有注册用户
- 编辑用户余额（HTP、USDT）
- 修改用户角色（普通用户/主节点/管理员）
- 管理钱包激活状态

### 2. 提币审核
- 查看所有提币申请
- 通过或拒绝提币申请

### 3. 主节点审核
- 查看所有主节点申请
- 通过或拒绝主节点申请

### 4. 开发者设置
- 查看当前开发者USDT地址
- 更新开发者钱包地址

## 📝 重要提示

1. **刷新页面**：修改角色后必须刷新页面或重新登录才能看到新权限
2. **权限隔离**：只有管理员才能看到管理后台入口，其他用户完全看不到
3. **安全性**：即使普通用户知道 `/admin` 路径，也会被路由守卫拦截
4. **第一个用户**：系统第一个注册的用户会自动成为管理员

## 🆘 如果还是看不到

如果刷新页面后仍然看不到管理后台入口：

1. **清除浏览器缓存**
   - Chrome: `Ctrl+Shift+Delete` → 清除缓存
   - 或者使用无痕模式重新登录

2. **检查用户信息**
   - 点击右上角用户头像
   - 确认显示的角色是"管理员"而不是"普通用户"或"主节点"

3. **重新登录**
   - 完全退出登录
   - 重新输入用户名和密码登录

4. **检查控制台**
   - 按 `F12` 打开开发者工具
   - 查看 Console 标签是否有错误信息
